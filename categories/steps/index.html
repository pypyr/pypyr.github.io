steps

Summary list of all ready-made steps in the pypyr estate.

titledescription
assertStop pipeline if item in context is not as expected.
callCall another step-group. Continue from the same place after the called groups complete.
cmdRun any external program, command, script.
contextclearRemove specified items from context.
contextclearallWipe the entire context.
contextmergeMerges values into context, preserving the existing context hierarchy.
contextsetCopy entire context keys.
contextsetfSet & format context keys.
debugPretty print context to output.
defaultSet values if they do not exist already.
echoEcho context value of echoMe to the output.
envGet, set or unset $ENVs.
envgetGet $ENVs & use a default fallback if they don’t exist.
fetchjsonLoad json file into pypyr context.
fetchyamlLoad yaml file into pypyr context.
fileformatFind & replace substitution {tokens} in any file.
fileformatjsonFind & replace substitution {tokens} in a json file.
fileformatyamlFind & replace substitution {tokens} in a yaml file.
filereplaceFind & replace any arbitrary search strings in a file.
filewritejsonWrite payload to file in json format.
filewriteyamlWrite payload to file in yaml format.
globGet paths from glob expression.
jumpJump to another step-group. The rest of the current step-group doesn’t run.
nowSaves current local date-time to context now.
nowutcSaves current utc date-time to context nowUtc.
pathcheckCheck if paths exist on filesystem.
pyExecute inline python code.
pypeCall another pipeline from the current pipeline.
pypyrversionWrite current pypyr version number to output.
safeshellDeprecated alias for cmd
shellRun commands in the default shell. Use for pipes, wildcards, $ENVs, ~
stepCreate your own custom step.
stopStop pypyr entirely.
stoppipelineStop current pipeline.
stopstepgroupStop current step-group.
tarArchive and/or extract tars with/without compression. Supports gzip, bzip2 & lzma.
